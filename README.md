# Personal Book Library Web Project 

Technologies:

- Java
- JUnit
- MySQL
- MongoDB
- Bootstrap CSS
- AngularJS
- Docker
- Spring Framework
- Spring Core
- Spring Data
- Spring MVC
- Spring Security


# Docker Commands

- docker pull mysql
- docker run --name mysqltest -p 3306:3306 -v ~/mysql_data:/var/lib/mysql  -e MYSQL_ROOT_PASSWORD=root -d mysql:latest

- docker pull mongo
- docker run -d -p 27017:27017 -v ~/mongodb_data:/data/db mongo
